
Tutorial
========

To get complete detail of the data structure and base classes used in ``SpyDrNet`` please refer
`byuccl/SpyDrNet Documentation <https://byuccl.github.io/spydrnet/docs/stable/index.html>`_
This page provides brief information of different classes concerning 
``SpyDrNet-Physical`` library.


Shell Interface
---------------

``SpyDrNet-Physical`` allows you to load a python shell-like interface after
installation with the command ``sdnphy``.
